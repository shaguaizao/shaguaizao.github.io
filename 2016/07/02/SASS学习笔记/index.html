SASS学习笔记

mac 安装

sudo gem install sass

使用

sass test.scss test.css

四种默认风格

  • nested：嵌套缩进，它是默认值。
  • expanded: 无缩进无扩展
  • compact: 简洁风格
  • compressed: 压缩后的，一般使用这项
